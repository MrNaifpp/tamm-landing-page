# Tamm Promotional Gifts Landing Page

## Architecture
- Single-page application (SPA) architecture.
- Responsive design using Tailwind CSS.
- RTL (Right-to-Left) layout for Arabic language support.

## Conventions
- **Styling:** Use Tailwind CSS utility classes. Avoid complex custom CSS unless necessary for unique animations.
- **Language:** Primary language is Arabic (ar).
- **Fonts:** Use 'Cairo' for body text, 'Kufam' for headings/branding, and 'Aref Ruqaa' for artistic taglines.
- **Interactions:** Use AOS (Animate On Scroll) for entry animations.

## Workflow
- Keep the single `index.html` structure unless it becomes too large to maintain.
- Ensure all new sections or components follow the existing aesthetic (Navy, Teal, Gold color palette).
